**1. (单选题) Er sucht ein neues Haus, denn er mag(喜欢) __________ altes Haus, er möchte __________ neues Haus kaufen.**
A kein, ein
B nicht, einen
C nicht, kein
D keinen, kein
* **答案：** A
* **解析：** Haus 是中性名词（das Haus）。第一空：名词前用 kein 否定，mag 后面接第四格（Akkusativ），中性第四格是 kein；第二空：表示“想买一栋新房子”，möchte 后面接第四格，中性第四格是不定冠词 ein。

**2. (单选题) __________ Sie bitte pünktlich!**
A Sein
B Sind
C Seien
D Ist
* **答案：** C
* **解析：** 考查 Sie（您）的命令格。动词 sein 针对 Sie 的命令格固定形式为 **Seien Sie!**（请您...）。

**3. (单选题) Der Zug __________ um 10 Uhr am Gleis 5 __________.**
A kommt, an
B ankommt, --
C kommt, --
D komme, an
* **答案：** A
* **解析：** 考查可分动词 ankommen（到达）。在现在时的一般陈述句中，副词前缀 an 必须放到句尾，变位动词 kommt 放在第二位（主语是单数第三人称 Der Zug）。

**4. (单选题) Der Arzt ist __________ das neue Medikament.**
A mit
B gegen
C um
D ohne
* **答案：** B
* **解析：** 句意为“医生**反对**这种新药”。gegen 是介词，意为“反对/违背”，后面直接接第四格（das Medikament 的第四格还是 das）。

**5. (单选题) Die Nachbarin hilft __________ Sohn bei den Hausaufgaben.**
A meiner
B meinen
C meine
D meinem
* **答案：** D
* **解析：** 动词 helfen（帮助）后面必须接第三格（Dativ）。Sohn（儿子）是阳性名词（der Sohn），它的单数第三格物主代词词尾是 `-em`，故选 meinem。

**6. (单选题) __________ du, wie spät es ist?**
A Weißt
B Wisst
C Weiß
D Wissen
* **答案：** A
* **解析：** 考查动词 wissen（知道）的变位。主语是 du，它的第二人称单数变位是 **weißt**（注意单数人称变位元音会由 i 变成 ei）。

**7. (单选题) Ein Fremder gibt __________ Eltern Blumen.**
A mein
B meine
C meinen
D meinem
* **答案：** C
* **解析：** 动词 geben 后面接双宾语（geben + 某人第三格 + 某物第四格）。Eltern（父母）是复数名词，复数第三格的物主代词词尾是 `-en`，故选 meinen。

**8. (单选题) Der __________ kommt aus Wien, und ich helfe dem __________ bei der Arbeit.**
A Herr, Herr
B Herr, Herrn
C Herrn, Herrn
D Herrn, Herr
* **答案：** B
* **解析：** 考查阳性弱变化名词 Herr（先生）。第一空作主语，用第一格原本形式 Herr；第二空在 dem 后面作第三格，Herr 作为弱变化名词必须加上词尾 `-n` 变成 **Herrn**。

**9. (单选题) Er ist krank, __________ er geht zur Arbeit.**
A aber
B denn
C oder
D und
* **答案：** A
* **解析：** 连词辨析。句意为：“他生病了，**但是**他去工作了。”表示转折用 aber。

**10. (单选题) Die Blumen gefallen __________ Tante gut, aber __________ nicht.**
A seiner, ihn
B seiner, ihm
C seine, ihm
D meinen, ihn
* **答案：** B
* **解析：** 动词 gefallen（喜欢/讨人喜欢）后面必须接第三格。Tante（姨妈/姑姑）是阴性名词，阴性第三格词尾是 `-er` 变成 seiner；第二空“但是不讨**他**喜欢”，er（他）的第三格是 **ihm**。

**11. (单选题) Seit __________ regnet es.**
A einem Tag
B einen Tag
C eine Tag
D ein Tag
* **答案：** A
* **解析：** 介词 seit（自从）后面必须接第三格（Dativ）。Tag（天）是阳性名词（der Tag），阳性单数第三格的不定冠词是 **einem**。

**12. (单选题) Morgen fliege ich __________ Hamburg, aber heute fahre ich __________ die Altstadt.**
A nach, in
B in, nach
C zu, in
D nach, zu
* **答案：** A
* **解析：** 地点介词辨析。第一空：飞往某个城市/国家名（无冠词）用 **nach** Hamburg；第二空：去老城区，Altstadt 是阴性名词（die Altstadt），表示进入某个空间用介词 in + 第四格（方向），阴性第四格依然是 **in die**。

**13. (单选题) In meinem Dorf __________ es __________ alten Zug.**
A gibt, einen
B gibt, eine
C gibt, einer
D gibt, einem
* **答案：** A
* **解析：** 固定句型 **Es gibt + 第四格**（有...）。Zug（火车）是阳性名词（der Zug），阳性第四格的不定冠词是 **einen**。

**14. (单选题) Er geht __________ seinen Regenschirm.**
A seit
B ohne
C gegen
D von
* **答案：** B
* **解析：** 介词辨析。句意为：“他**没带**雨伞就出门了。”ohne 意为“没有/不带”，后面固定接第四格（seinen Regenschirm 正好是阳性第四格形式）。

**15. (单选题) Er __________ den Text ohne Erklärung.**
A versteht
B steht ver
C verstand
D verstehtet
* **答案：** A
* **解析：** 考查不可分动词 verstehen（理解）的现在时变位。主语是 Er（单三），直接在词尾加 `-t` 变成 versteht（ver- 是不可分前缀，绝对不能拆到句尾）。

**16. (单选题) Wir __________ gern ein neues Auto kaufen, aber wir haben nicht genug Geld.**
A möchtet
B möchte
C möchten
D möchtest
* **答案：** C
* **解析：** 考查情态动词 möchten 的变位。主语是 Wir（我们），第一人称复数对应的正确变位是 **möchten**。

**17. (单选题) Der Arzt hilft __________ bei der Gesundheitsprüfung.**
A einem Franzosen
B einen Franzosen
C einem Franzose
D einen Franzos
* **答案：** A
* **解析：** 动词 helfen 后面接第三格。Franzose（法国男人）是一个阳性弱变化名词，只要不是第一格，其他所有格都要加 `-n` 变成 Franzosen。加上阳性单数第三格不定冠词 einem，组合成 **einem Franzosen**。

**18. (单选题) Der Professor zeigt __________ Studenten __________ neuen Plan.**
A seinem, eine
B seinen, einem
C seine, einen
D seinem, den
* **答案：** D
* **解析：** 动词 zeigen 后面接双宾语（显示某人第三格 + 某物第四格）。Student（学生）是阳性弱变化名词，单数第三格不仅前面要用 seinem，词尾也要加 `-en`，变成 **seinem** Studenten；Plan（计划）也是阳性名词（der Plan），它的单数第四格定冠词是 **den**。

**19. (单选题) __________ Sie bitte geduldig!**
A Sein
B Sind
C Seien
D Ist
* **答案：** C
* **解析：** 考查 Sie 的命令格（与第 2 题完全相同）。`Seien Sie bitte geduldig!` 意为“请您耐心一点！”

**20. (单选题) Mein Bruder __________ einen Hund. Er __________ klein.**
A hat, ist
B hat, bin
C haben, ist
D habt, ist
* **答案：** A
* **解析：** 第一空：我的哥哥有一条狗，主语 Mein Bruder 是单数第三人称，haben 变位用 **hat**；第二空：它（狗）很小，der Hund 用代词 er 代替，er 对应的 sein 动词变位是 **ist**。

# 德语单选题练习与解析（介词专项，共10题）

**1. (单选题) Deine Schwester kommt gerade ______ dem Zimmer.**
A. aus
B. für
C. um
D. mit
*   **正确答案：** A
*   **解析：** 考查表示方向和位置的介词。`aus` + 第三格（Dativ）表示“从……（封闭的空间）里出来”。句意为：“你妹妹正从房间里出来。”

**2. (单选题) Das ist das Foto ______ meiner Tante.**
A. zu
B. von
C. auf
D. in
*   **正确答案：** B
*   **解析：** `von` + 第三格（Dativ）可以用来表示所属关系，相当于英文中的 "of" 或汉语中的“的”。句意为：“这是我阿姨的照片。”

**3. (单选题) Sie wohnt ______ ihrer Großmutter.**
A. an
B. nach
C. bei
D. vor
*   **正确答案：** C
*   **解析：** `bei` + 第三格（Dativ）常用于表示“住在某人那里”或“在某人家里”。句意为：“她和她的祖母住在一起。”

**4. (单选题) Er schreibt ______ einem Kugelschreiber.**
A. in
B. zu
C. bei
D. mit
*   **正确答案：** D
*   **解析：** `mit` + 第三格（Dativ）可以用来表示使用某种工具或手段。句意为：“他用圆珠笔写字。”

**5. (单选题) Wir fahren morgen ______ München.**
A. auf
B. nach
C. aus
D. vor
*   **正确答案：** B
*   **解析：** 表示前往某个城市、国家（无冠词）或大洲等地理名称时，介词使用 `nach`。句意为：“我们明天去慕尼黑。”

**6. (单选题) Ich gehe gerne ______ dir ins Kino.**
A. für
B. um
C. mit
D. zu
*   **正确答案：** C
*   **解析：** `mit` + 第三格（Dativ）表示伴随，意为“和……一起”。`dir` 是 `du` 的第三格形式。句意为：“我喜欢和你一起去看电影。”

**7. (单选题) ______ dem Unterricht spielen wir Fußball.**
A. Auf
B. Zu
C. Um
D. Nach
*   **正确答案：** D
*   **解析：** `nach` + 第三格（Dativ）在表示时间时，意为“在……之后”。句意为：“下课后我们去踢足球。”

**8. (单选题) Herr Li kommt ______ Japan.**
A. von
B. zu
C. mit
D. aus
*   **正确答案：** D
*   **解析：** 表示“从某个国家/地方来”（籍贯、出生地或出发地），固定搭配是 `kommen aus` + 第三格（Dativ）。句意为：“李先生来自日本。”

**9. (单选题) Sie arbeitet ______ einer großen Firma.**
A. mit
B. nach
C. bei
D. von
*   **正确答案：** C
*   **解析：** 表示“在某家公司、企业或机构工作”时，使用的介词是 `bei` + 第三格（Dativ）。句意为：“她在一家大公司工作。”

**10. (单选题) Meine Schwester lernt ______ zwei Klavier.** *(注：原题干中的 "zwei" 后面大概率遗漏了时间名词，如 Jahren)*
A. für
B. um
C. zu
D. seit
*   **正确答案：** D
*   **解析：** 考查时间介词。`seit` + 第三格（Dativ）表示动作从过去某一时刻开始并一直延续到现在，意为“自从……以来”或“持续了……（时间）”。句意为：“我妹妹学钢琴已经两（年/个月）了。” 虽然原题干在 "zwei" 后面漏掉了名词（如 `Jahren`），但结合动词的现在时态（lernt）和延续性动作，正确答案毫无疑问是表示持续时间的 `seit`。

# 德语基础语法变格练习题（表格改编版，共20题）

## 第一部分：人称代词 (Personalpronomen)

**1. (单选题) 德语人称代词“我 (ich)”的主格(N)、宾格(A)和与格(D)依次是：**
A. ich, mir, mich
B. ich, mich, mir
C. mich, ich, mir
D. mir, mich, ich
*   **正确答案：** B
*   **解析：** “我”的主格是 ich，宾格是 mich，与格是 mir。记忆口诀：宾格 m**i**ch 对应 d**i**ch，与格 m**i**r 对应 d**i**r。

**2. (单选题) 德语人称代词“你 (du)”的主格(N)、宾格(A)和与格(D)依次是：**
A. du, dich, dir
B. du, dir, dich
C. dich, du, dir
D. dir, dich, du
*   **正确答案：** A
*   **解析：** “你”的主格是 du，宾格是 dich，与格是 dir。

**3. (单选题) 德语人称代词“阳性 ta (er)”、“阴性 ta (sie)”、“中性 ta (es)”的宾格(A)依次是：**
A. ihn, sie, es
B. ihm, ihr, ihm
C. er, sie, es
D. ihn, ihr, es
*   **正确答案：** A
*   **解析：** 宾格下，只有阳性 er 会变成 ihn，阴性 sie 和中性 es 的宾格与主格保持一致（不变）。B 选项是这三者的与格。

**4. (单选题) 德语人称代词“我们 (wir)”和“你们 (ihr)”的与格(D)依次是：**
A. uns, euch
B. euch, uns
C. wir, ihr
D. uns, ihnen
*   **正确答案：** A
*   **解析：** wir 的宾格和与格都是 uns；ihr 的宾格和与格都是 euch。

**5. (单选题) 德语尊称“您/您们 (Sie)”的宾格(A)和与格(D)依次是：**
A. Sie, Ihnen
B. sie, ihnen
C. Ihnen, Sie
D. ihnen, sie
*   **正确答案：** A
*   **解析：** 尊称 Sie 的宾格还是 Sie，与格是 Ihnen。注意首字母必须大写，以区别于复数第三人称 sie。

## 第二部分：否定词 (kein)

**6. (单选题) 否定词 kein 的中性(n.)和阳性(m.)的与格(D)形式分别是：**
A. keinem, keinem
B. keinen, keinem
C. keinem, keinen
D. keinen, keinen
*   **正确答案：** A
*   **解析：** 在德语变格中，中性和阳性的与格（Dativ）词尾通常是相同的，都是加 -em，所以都是 keinem。

**7. (单选题) 否定词 kein 的阴性(f.)主格(N)、宾格(A)、与格(D)依次是：**
A. keine, keine, keinen
B. keine, keine, keiner
C. kein, kein, keiner
D. keine, keiner, keinen
*   **正确答案：** B
*   **解析：** 阴性词的主格和宾格相同，都是 keine；阴性与格词尾加 -er，变成 keiner。

**8. (单选题) 否定词 kein 的复数(pl.)主格(N)、宾格(A)、与格(D)依次是：**
A. keine, keine, keinen
B. keine, keine, keiner
C. keinen, keinen, keine
D. keine, keinen, keinen
*   **正确答案：** A
*   **解析：** 复数的主格和宾格相同，都是 keine；复数与格词尾加 -en，变成 keinen（且名词复数后通常还要加 -n）。

**9. (单选题) 否定词 kein 的阳性(m.)主格(N)、宾格(A)、与格(D)依次是：**
A. kein, keinen, keinem
B. kein, keinem, keinen
C. kein, kein, keinem
D. keinen, kein, keinem
*   **正确答案：** A
*   **解析：** 阳性主格为 kein，宾格加 -en 变为 keinen，与格加 -em 变为 keinem。

## 第三部分：定冠词 (Bestimmter Artikel)

**10. (单选题) 定冠词的阳性(m.)主格(N)、宾格(A)、与格(D)依次是：**
A. der, den, dem
B. der, dem, den
C. den, der, dem
D. dem, den, der
*   **正确答案：** A
*   **解析：** 阳性定冠词变格规律：主格 der，宾格 den，与格 dem。

**11. (单选题) 定冠词的阴性(f.)和复数(pl.)的主格(N)和宾格(A)都是：**
A. der
B. die
C. das
D. den
*   **正确答案：** B
*   **解析：** 阴性单数和复数的第一格（主格）和第四格（宾格）共用同一个定冠词 die。

**12. (单选题) 定冠词的中性(n.)主格(N)、宾格(A)、与格(D)依次是：**
A. das, das, dem
B. das, dem, das
C. das, das, den
D. dem, das, das
*   **正确答案：** A
*   **解析：** 中性定冠词变格规律：主格和宾格都是 das，与格和阳性一样，是 dem。

**13. (单选题) 定冠词的复数(pl.)与格(D)是：**
A. der
B. die
C. den
D. dem
*   **正确答案：** C
*   **解析：** 复数与格定冠词是 den（且注意其后的名词复数形式通常需要加上词尾 -n）。

## 第四部分：不定冠词 (ein)

**14. (单选题) 不定冠词 ein 的阴性(f.)主格(N)、宾格(A)、与格(D)依次是：**
A. eine, eine, einer

C. ein, eine, einer
D. eine, eine, einem
*   **正确答案：** A
*   **解析：** 不定冠词的阴性变位与 keine 完全一致：主格和宾格都是 eine，与格是 einer。

**15. (单选题) 不定冠词 ein 的阳性(m.)主格(N)、宾格(A)、与格(D)依次是：**
A. ein, einem, einen
B. ein, einen, einem
C. einen, ein, einem
D. ein, ein, einem
*   **正确答案：** B
*   **解析：** 阳性主格为 ein，宾格加 -en 变为 einen，与格加 -em 变为 einem。

**16. (单选题) 不定冠词 ein 的中性(n.)主格(N)、宾格(A)、与格(D)依次是：**
A. ein, ein, einem
B. ein, einem, ein
C. ein, ein, einen
D. einem, ein, ein
*   **正确答案：** A
*   **解析：** 中性主格和宾格相同，都是 ein，与格与阳性相同，是 einem。

## 第五部分：物主代词 (euer)

**17. (单选题) 物主代词 euer（你们的）在阴性(f.)和复数(Pl.)的主格(N)与宾格(A)形式是：**
A. euer
B. euere
C. eure
D. euren
*   **正确答案：** C
*   **解析：** 当 euer 后面需要加词尾 -e 时，词干中间的 e 必须省略，因此不能写成 euere，必须写成 eure。

**18. (单选题) 物主代词 euer 的阳性(m.)主格(N)、宾格(A)、与格(D)依次是：**
A. euer, euren, eurem
B. euer, eurem, euren
C. eure, euren, eurem
D. euer, euren, euer
*   **正确答案：** A
*   **解析：** 阳性主格是原型 euer；宾格加 -en 时省 e 变为 euren；与格加 -em 时省 e 变为 eurem。

**19. (单选题) 物主代词 euer 的中性(n.)主格(N)、宾格(A)、与格(D)依次是：**
A. euer, euer, euren
B. euer, euer, eurem
C. eure, euer, eurem
D. euer, euren, eurem
*   **正确答案：** B
*   **解析：** 中性主格和宾格都是原型 euer（不需要加词尾，所以不省略 e）；与格和阳性一样，是 eurem。

**20. (单选题) 物主代词 euer 的复数(Pl.)与格(D)是：**
A. eurer
B. eurem
C. euren
D. eure
*   **正确答案：** C
*   **解析：** 复数与格的词尾是 -en，所以 euer 加 -en 并在词干省 e 后，变成 euren。
