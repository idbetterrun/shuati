依据《德语2-语法点总结》Teil A-D 整理，涵盖：动词变位、祈使句、可分/不可分动词、haben变位、四格与三格系统、介词、双宾语语序、弱变化名词、并列连词等。

## Teil A：动词现在时变位 与 祈使句（第1-12题）

### 一、不规则动词变位（1-6）

**1. (单选题) Meine Schwester __________ (sprechen) drei Fremdsprachen.**
A spricht
B sprecht
C sprichst
D sprechen
* **答案：** A
* **解析：** sprechen 属于 e→i 变元音动词，只有 du 和 er/sie/es 两个人称变元音。主语是第三人称单数 Schwester，故用 spricht。

**2. (单选题) __________ du gern Schokolade? (essen)**
A Isst
B Esst
C Issst
D Essen
* **答案：** A
* **解析：** essen 是 e→i 变化动词，du 的形式为 isst（essen 词干 ess- + st，因词干已以 s 结尾，与 st 合并为 isst，只写一个 s）。B 是 ihr 的形式，需排除。

**3. (单选题) Mein Vater __________ (lesen) jeden Abend Zeitung.**
A liest
B lest
C liesst
D lesen
* **答案：** A
* **解析：** lesen 是 e→ie 变化动词，er/sie/es 形式为 liest（与 du 同形）。C 拼写错误（多写了一个 s），D 是原形/wir/sie复数形式。

**4. (单选题) __________ du schon? (schlafen)**
A Schlafst
B Schläfst
C Schlaft
D Schläft
* **答案：** B
* **解析：** schlafen 是 a→ä 变化动词，du 形式为 schläfst。C 是普通拼写但缺少元音变化，D 是 er/sie/es 的形式，人称不对。

**5. (单选题) Der Junge __________ (laufen) sehr schnell.**
A laufst
B loft
C läuft
D laufen
* **答案：** C
* **解析：** laufen 属于 au→äu 变化动词，er/sie/es 形式为 läuft。

**6. (单选题) Was __________ du zum Frühstück? (nehmen)**
A nehmst
B nimmst
C nimmt
D nehmt
* **答案：** B
* **解析：** nehmen 是特殊的 e→i 变化动词，同时伴随辅音变化（nehm- → nimm-），du 形式为 nimmst，而不是规则的 nehmst。C 是 er/sie/es 形式，人称不对。

### 二、wissen的变位（7-8）

**7. (单选题) Ich __________ (wissen) die Lösung nicht.**
A weiß
B weißt
C wisse
D weiss
* **答案：** A
* **解析：** wissen 单数变位类似情态动词：ich weiß, du weißt, er/sie/es weiß。D 的 "weiss" 是错误拼写（应使用 ß，而非 ss）。

**8. (单选题) __________ ihr, wo das Museum ist? (wissen)**
A Wisst
B Weißt
C Wisse
D Wissen
* **答案：** A
* **解析：** wissen 的 ihr 变位为 wisst（复数人称不变元音，规则变位）。B 是 du 的形式，D 是原形/wir/sie复数形式。

### 三、祈使句（9-12）

**9. (单选题) (du) ___________ das Fenster, bitte! (öffnen)**
A Öffn
B Öffne
C Öffnst
D Öffnet
* **答案：** B
* **解析：** du 式祈使句：去掉 du，用 du 变位形式去掉 -st。öffnen 词干以 -n 结尾，需保留 -e：Öffne！A 缺少必要的 -e，D 是 ihr 的形式。

**10. (单选题) (du) ___________ dein Gemüse! (essen)**
A Ess
B Iss
C Esst
D Isst
* **答案：** B
* **解析：** essen 是 e→i 变化动词，祈使句中保留元音变化：du isst → 去掉 -st → Iss！A 没有保留元音变化，错误；D 保留了 -st 词尾，未去掉。

**11. (单选题) 选出正确的祈使句（du, fahren - 请小心驾驶！）**
A Fahr vorsichtig!
B Fähr vorsichtig!
C Fahrt vorsichtig!
D Führ vorsichtig!
* **答案：** A
* **解析：** fahren 的 a→ä 变化在祈使句中不适用，所以是 Fahr! 而不是 Fähr!。C 是 ihr 的形式，D 是不相关的动词 führen。

**12. (单选题) (Sie) 请进来！___________ bitte herein! (kommen)**
A Sie kommen
B Kommen Sie
C Komm Sie
D Kommt Sie
* **答案：** B
* **解析：** Sie 式祈使句：动词提到最前面，加 Sie，即"动词+Sie"的倒装结构：Kommen Sie!

## Teil B：可分动词与不可分动词（第13-22题）

**13. (单选题) 选出语序正确的句子（aufstehen, ich - 我周末总是起得很晚）**
A Ich stehe auf am Wochenende immer spät.
B Ich stehe am Wochenende immer spät auf.
C Ich aufstehe am Wochenende immer spät.
D Am Wochenende ich stehe immer spät auf.
* **答案：** B
* **解析：** 可分动词构成框架结构：变位动词占第二位，前缀 auf 永远在句尾。D 虽把 Am Wochenende 放第一位，但违反了动词第二位规则（谓语应紧跟其后）。

**14. (单选题) 选出语序正确的疑问句（fernsehen, du - 你今晚看电视吗？）**
A Du siehst heute Abend fern?
B Siehst du heute Abend fern?
C Fernsiehst du heute Abend?
D Siehst fern du heute Abend?
* **答案：** B
* **解析：** 一般疑问句中变位动词提到第一位，前缀依然留在句尾：Siehst du heute Abend fern?

**15. (单选题) 选出语序正确的特殊疑问句（wann, aufstehen, er - 他什么时候起床？）**
A Wann er steht auf?
B Wann steht er auf?
C Wann aufsteht er?
D Wann steht auf er?
* **答案：** B
* **解析：** 特殊疑问句：疑问词占第一位，变位动词占第二位，前缀依然放句尾：Wann steht er auf?

**16. (单选题) 选出正确的句子（einladen, er - 他邀请朋友们参加聚会，注意元音变化）**
A Er einlädt seine Freunde zur Party.
B Er lädt seine Freunde zur Party ein.
C Er ladet seine Freunde zur Party ein.
D Er lädt ein seine Freunde zur Party.
* **答案：** B
* **解析：** einladen 是可分动词，词干 laden 本身 a→ä 变化（er lädt），变位后前缀 ein 分离放句尾：Er lädt seine Freunde zur Party ein.

**17. (单选题) 选出正确的疑问句（mitnehmen, du - 你带雨伞了吗？注意元音变化）**
A Nimmst du deinen Regenschirm mit?
B Nehmst du deinen Regenschirm mit?
C Mitnimmst du deinen Regenschirm?
D Nimmt du deinen Regenschirm mit?
* **答案：** A
* **解析：** mitnehmen 的词干 nehmen 本身有 e→i+辅音变化（du nimmst），可分前缀规则同时适用，前缀 mit 放句尾：Nimmst du deinen Regenschirm mit?

**18. (单选题) 下列哪个动词是不可分动词？**
A aufstehen
B verstehen
C einkaufen
D mitkommen
* **答案：** B
* **解析：** ver- 是不可分前缀（常见不可分前缀：be-, ge-, er-, ver-, zer-, ent-, emp-, miss-），重音不在前缀上，动词整体不分离。其余三个都是可分动词。

**19. (单选题) 选出正确的疑问句（verstehen, ihr - 你们理解这个任务吗？）**
A Versteht ihr die Aufgabe?
B Verstehen ihr die Aufgabe?
C Verstehst ihr die Aufgabe?
D Versteht die Aufgabe ihr?
* **答案：** A
* **解析：** verstehen 虽是不可分动词，但仍需正常变位，ihr 的形式是 versteht，不能用原形 verstehen 或 du 的 verstehst。

**20. (单选题) 选出正确的句子（besuchen, wir - 我们每年拜访祖父母）**
A Wir besuchen unsere Großeltern jedes Jahr.
B Wir besucht unsere Großeltern jedes Jahr.
C Wir besuchen jedes Jahr unsere Großeltern besuchen.
D Besuchen wir unsere Großeltern jedes Jahr besuchen.
* **答案：** A
* **解析：** besuchen 不可分，wir 的正常变位是 besuchen；C、D 重复出现动词，属于病句。

**21. (单选题) 选出正确的句子（vergessen - 我忘记了我的书）**
A Ich vergesse mein Buch.
B Ich esse mein Buch ver.
C Ich vergesse mein Buch ver.
D Vergesse ich mein Buch.
* **答案：** A
* **解析：** vergessen 中的 ver- 是不可分前缀，动词整体变位、不拆分：Ich vergesse mein Buch.

**22. (单选题) 选出正确的疑问句（mitkommen, du - 你为什么不一起来？）**
A Warum kommst du nicht mit?
B Warum du kommst nicht mit?
C Warum nicht kommst du mit?
D Warum mitkommst du nicht?
* **答案：** A
* **解析：** mitkommen 是可分动词，疑问词 warum 占第一位，变位动词 kommst 占第二位，nicht 放在前缀 mit 之前，前缀留句尾：Warum kommst du nicht mit?

## Teil C：haben的现在时变位（第23-27题）

**23. (单选题) Ich __________ keine Zeit. (haben)**
A habe
B hast
C hat
D haben
* **答案：** A
* **解析：** haben 的 ich 变位是 habe。

**24. (单选题) __________ du Geschwister? (haben)**
A Hast
B Habst
C Hat
D Habest
* **答案：** A
* **解析：** haben 的 du 变位是 hast（不规则，非 habst）。

**25. (单选题) Er __________ ein neues Auto. (haben)**
A habt
B hat
C habst
D hätt
* **答案：** B
* **解析：** haben 的 er/sie/es 变位是 hat。

**26. (单选题) __________ ihr Hunger? (haben)**
A Habt
B Habet
C Hast
D Haben
* **答案：** A
* **解析：** haben 的 ihr 变位是 habt。

**27. (单选题) Sie(他们) __________ viele Freunde. (haben)**
A hat
B habt
C haben
D habst
* **答案：** C
* **解析：** haben 的 sie(复数)/Sie 变位是 haben（与原形同形）。

## Teil D：格系统（四格/三格）及相关语法点（第28-69题）

### 一、四格 - 动词直接宾语（28-30）

**28. (单选题) Ich kaufe __________. (ein Buch - 一本书)**
A ein Buch
B einen Buch
C eine Buch
D dem Buch
* **答案：** A
* **解析：** das Buch 是中性名词，中性名词在第四格时不定冠词形式不变：ein Buch。

**29. (单选题) Ich sehe __________ jeden Tag. (der Lehrer - 那位老师)**
A der Lehrer
B die Lehrer
C den Lehrer
D dem Lehrer
* **答案：** C
* **解析：** sehen 的直接宾语用第四格，der Lehrer 是阳性名词，定冠词第四格为 den。

**30. (单选题) Wir brauchen __________. (die Hilfe - 帮助)**
A die Hilfe
B der Hilfe
C den Hilfe
D das Hilfe
* **答案：** A
* **解析：** die Hilfe 是阴性名词，阴性定冠词第四格与第一格相同，仍是 die。

### 二、四格介词：durch, für, gegen, ohne, um, bis（31-35）

**31. (单选题) Das Geschenk ist für __________ Bruder. (mein - 我的哥哥)**
A mein
B meinen
C meinem
D meiner
* **答案：** B
* **解析：** für 是固定接第四格的介词，der Bruder 是阳性名词，物主代词第四格阳性词尾为 -en：meinen。

**32. (单选题) Er kämpft gegen __________ Feind. (der - 敌人)**
A der
B den
C dem
D des
* **答案：** B
* **解析：** gegen 固定接第四格，der Feind 阳性名词，定冠词第四格为 den。

**33. (单选题) Sie geht ohne __________ Mantel aus dem Haus. (ihr - 她的外套)**
A ihr
B ihre
C ihren
D ihrem
* **答案：** C
* **解析：** ohne 固定接第四格，der Mantel 阳性名词，物主代词第四格阳性词尾为 -en：ihren。

**34. (单选题) Wir laufen um __________ See. (der See - 湖)**
A der
B den
C dem
D des
* **答案：** B
* **解析：** um 固定接第四格，der See 阳性名词，定冠词第四格为 den。

**35. (单选题) Bis __________ Montag habe ich Zeit. (nächst- - 下周一)**
A nächstes
B nächsten
C nächstem
D nächster
* **答案：** B
* **解析：** bis 固定接第四格，der Montag 阳性名词，无冠词时形容词强变化阳性第四格词尾为 -en：nächsten Montag。

### 三、四格特殊结构（时间段/es gibt等）（36-38）

**36. (单选题) __________ Morgen trinke ich Kaffee. (jed- - 每天早上)**
A Jeder
B Jeden
C Jedem
D Jedes
* **答案：** B
* **解析：** 表示"每天早上"这类时间点用第四格结构，der Morgen 阳性名词，第四格词尾为 -en：Jeden Morgen。

**37. (单选题) Es gibt __________ Problem. (kein - 没有问题)**
A kein
B keinen
C keinem
D keine
* **答案：** A
* **解析：** es gibt 固定接第四格，das Problem 是中性名词，kein 中性第四格不加词尾：kein。

**38. (单选题) Ich bleibe __________ Woche in Berlin. (ein - 一周)**
A ein
B eine
C einer
D einen
* **答案：** B
* **解析：** 表示持续时间段（"我在柏林待一周"）用第四格，die Woche 是阴性名词，不定冠词阴性第四格不变：eine。

**40. (单选题) Sie liebt __________ Katze. (ihr - 她的猫)**
A ihr
B ihre
C ihren
D ihrer
* **答案：** B
* **解析：** die Katze 是阴性名词，物主代词阴性第四格与第一格相同：ihre。

**41. (单选题) Er trinkt __________ Bier. (kein - 不喝啤酒)**
A kein
B keine
C keinen
D keinem
* **答案：** A
* **解析：** das Bier 是中性名词，kein 中性第四格不加词尾：kein。

### 五、人称代词四格（42-44）

**42. (单选题) Ich liebe __________. (她)**
A sie
B ihr
C ihn
D ihre
* **答案：** A
* **解析：** "她"的第四格人称代词是 sie（第三格才是 ihr）。

**43. (单选题) Kennst du __________? (我)**
A mir
B mich
C mein
D meiner
* **答案：** B
* **解析：** "我"的第四格人称代词是 mich（第三格是 mir）。

**44. (单选题) Wir besuchen __________ morgen. (你们)**
A euch
B ihr
C euer
D ihnen
* **答案：** A
* **解析：** "你们"的第四格人称代词是 euch（第三格也是 euch，与第四格同形；ihr 是第一格）。

### 六、三格形式：定冠词/不定冠词/物主代词（45-47）

**45. (单选题) Ich helfe __________ Mann. (der - 那位男士)**
A der
B den
C dem
D des
* **答案：** C
* **解析：** helfen 是必须接第三格的动词，der Mann 阳性名词，定冠词第三格为 dem。

**46. (单选题) Sie gibt __________ Kind ein Geschenk. (ein - 一个小孩)**
A ein
B einem
C einen
D eines
* **答案：** B
* **解析：** geben 的间接宾语用第三格，das Kind 中性名词，不定冠词第三格为 einem。

**47. (单选题) Er dankt __________ Frau. (sein - 他的妻子)**
A sein
B seine
C seiner
D seinem
* **答案：** C
* **解析：** danken 是必须接第三格的动词，die Frau 阴性名词，物主代词第三格阴性词尾为 -er：seiner。

### 七、三格介词：aus, bei, mit, nach, von, zu, seit（48-52）

**48. (单选题) Ich komme aus __________ Schweiz. (die - 瑞士)**
A die
B der
C dem
D den
* **答案：** B
* **解析：** aus 固定接第三格，die Schweiz 是阴性国名（需带冠词），定冠词第三格阴性为 der：aus der Schweiz。

**49. (单选题) Er wohnt bei __________ Eltern. (sein - 他的父母)**
A sein
B seine
C seinen
D seiner
* **答案：** C
* **解析：** bei 固定接第三格，Eltern 是复数名词，物主代词第三格复数词尾为 -en：seinen。

**50. (单选题) Wir fahren mit __________ Zug. (der - 火车)**
A der
B den
C dem
D des
* **答案：** C
* **解析：** mit 固定接第三格，der Zug 阳性名词，定冠词第三格为 dem。

**51. (单选题) Nach __________ Essen gehen wir spazieren. (das - 晚饭)**
A das
B dem
C der
D den
* **答案：** B
* **解析：** nach 固定接第三格，das Essen 中性名词，定冠词第三格为 dem。

**52. (单选题) Seit __________ Woche lernt er Deutsch. (ein - 一周)**
A ein
B eine
C einer
D einem
* **答案：** C
* **解析：** seit 固定接第三格，die Woche 阴性名词，不定冠词第三格阴性为 einer。

### 八、möchten的变位及用法（53-55）

**53. (单选题) Ich __________ einen Kaffee. (möchten)**
A möchte
B möchtest
C möchtet
D mögt
* **答案：** A
* **解析：** möchten 的 ich 变位是 möchte，表示委婉的"想要"。

**54. (单选题) __________ du auch etwas essen? (möchten)**
A Möchte
B Möchtest
C Möchtet
D Mögst
* **答案：** B
* **解析：** möchten 的 du 变位是 möchtest。

**55. (单选题) Was __________ ihr trinken? (möchten)**
A möchte
B möchtest
C möchtet
D möchten
* **答案：** C
* **解析：** möchten 的 ihr 变位是 möchtet；且 möchten 作为情态动词，句末需要接动词原形 trinken（框架结构）。

### 九、三格中双宾语的位置（56-57）

**56. (单选题) 选出正确语序（送我妈妈一本书，两个都是名词）**
A Ich schenke ein Buch meiner Mutter.
B Ich schenke meiner Mutter ein Buch.
C Ich schenke meiner ein Buch Mutter.
D Meiner Mutter schenke ein Buch ich.
* **答案：** B
* **解析：** 当第三格和第四格宾语都是名词时，语序规则是：第三格在前，第四格在后。

**57. (单选题) 选出正确语序（把它给我父亲，es 是四格代词）**
A Ich gebe es meinem Vater.
B Ich gebe meinem Vater es.
C Ich gebe meinem es Vater.
D Es gebe ich meinem Vater.
* **答案：** A
* **解析：** 当其中一个宾语是代词、另一个是名词时，代词永远优先放在名词前面，不论代词是第几格：Ich gebe es meinem Vater.

### 十、三格中复数名词的变化（58-59）

**58. (单选题) Die Lehrerin hilft __________. (die Kinder - 孩子们)**
A die Kinder
B den Kinder
C den Kindern
D der Kindern
* **答案：** C
* **解析：** 名词复数在第三格时，若不是以 -n 或 -s 结尾，需加 -n：die Kinder → den Kindern。

**59. (单选题) Er spielt mit __________. (die Autos - 玩具车)**
A den Autos
B den Autosn
C den Auton
D der Autos
* **答案：** A
* **解析：** die Autos 复数已经以 -s 结尾，第三格时不需要再加 -n：den Autos。

### 十一、弱变化名词（60-62）

**60. (单选题) Ich sehe __________. (der Student - 那个学生)**
A der Student
B den Student
C den Studenten
D dem Studenten
* **答案：** C
* **解析：** der Student 是弱变化名词，除第一格单数外，其余格都要加 -en：第四格为 den Studenten。

**61. (单选题) Sie hilft __________. (der Junge - 那个男孩)**
A der Junge
B den Junge
C dem Junge
D dem Jungen
* **答案：** D
* **解析：** der Junge 是弱变化名词，第三格也要加 -n：dem Jungen。

**62. (单选题) 下列哪个名词不属于弱变化名词(n-Deklination)？**
A der Student
B der Mensch
C der Herr
D der Tisch
* **答案：** D
* **解析：** der Tisch 是普通阳性名词，按常规变格；der Student, der Mensch, der Herr 都是常见的弱变化名词。

### 十二、并列连词 aber, und, denn, oder 不占位（63-65）

**63. (单选题) 选出正确语序（Ich lerne Deutsch, ... er lernt Englisch.）**
A Ich lerne Deutsch, und lernt er Englisch.
B Ich lerne Deutsch, und er lernt Englisch.
C Ich lerne Deutsch, und Englisch lernt er.
D Ich lerne Deutsch, und er Englisch lernt.
* **答案：** B
* **解析：** und 不占句子位置，后面从句依然遵循主语+变位动词的正常语序：und er lernt Englisch。

**64. (单选题) 选出正确语序（Ich bleibe zu Hause, ... ich bin krank.）**
A Ich bleibe zu Hause, denn bin ich krank.
B Ich bleibe zu Hause, denn ich bin krank.
C Ich bleibe zu Hause, denn ich krank bin.
D Ich bleibe zu Hause, denn krank bin ich.
* **答案：** B
* **解析：** denn 不占位，其后从句正常语序（主语+变位动词）：denn ich bin krank。

**65. (单选题) 以下哪个词会占据第一位、导致动词提前，与 aber, und, denn, oder 不同？**
A oder
B deshalb
C und
D aber
* **答案：** B
* **解析：** deshalb 是副词，会占据句子第一位并引发动词提前（V2语序）；而 aber, und, denn, oder 是并列连词，不占位，不影响其后从句的语序。

### 十三、必须接三格的动词（66-69）

**66. (单选题) Ich helfe __________ jeden Tag. (mein Bruder - 我哥哥)**
A mein Bruder
B meinen Bruder
C meinem Bruder
D meines Bruders
* **答案：** C
* **解析：** helfen 是必须接第三格的动词，der Bruder 阳性名词，第三格为 meinem Bruder。

**67. (单选题) Das Kleid gefällt __________ sehr gut. (ich - 我)**
A ich
B mich
C mir
D meiner
* **答案：** C
* **解析：** gefallen 是必须接第三格的动词，"我"的第三格人称代词是 mir。

**68. (单选题) __________ gehört dieses Auto? (wer的三格形式)**
A Wer
B Wen
C Wem
D Wessen
* **答案：** C
* **解析：** gehören 是必须接第三格的动词，"谁"的第三格疑问代词是 wem。

**69. (单选题) Ich danke __________ herzlich. (Sie - 您，敬语)**
A Sie
B Ihnen
C Ihr
D Ihre
* **答案：** B
* **解析：** danken 是必须接第三格的动词，敬语"您"的第三格人称代词是 Ihnen（Sie 是第一/四格形式）。
